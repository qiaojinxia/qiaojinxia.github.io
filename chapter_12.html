Leetcode 第70题 爬楼梯

假设你正在爬楼梯。需要 n 阶你才能到达楼顶。

每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？

**注意：**给定 n 是一个正整数。

示例 1：

输入： 2 输出： 2 解释： 有两种方法可以爬到楼顶。

  1. 1 阶 + 1 阶
  2. 2 阶

示例 2：

输入： 3 输出： 3 解释： 有三种方法可以爬到楼顶。

  1. 1 阶 + 1 阶 + 1 阶
  2. 1 阶 + 2 阶
  3. 2 阶 + 1 阶

解题思路

这题 就是求解斐波那契数, 1 1 2 我们可以使用递归的方式,但是时间复杂度 会非常高,因为 每次递归包含了一些重复的 计算,所以 我们优化的思维是,计算过的 爬第 n阶台阶,我们可以 用O(n)空间去存放,所有之前计算过的台阶。

进一步来讲,我们 可以 把问题 看成

一个组合问题

走一阶台阶 { 1阶 }

走两阶台阶 {1阶 1阶 、 2阶 }

走三阶台阶 {1阶 1阶 1阶 、 2阶 1阶、3阶 }

这样 就变成 组合问题了,拿走三阶为例 我们可以将问转化成 走了 2阶条件下,走三阶该怎么走。

这就是动态规划的思想。

代码

pub fn climb_stairs(n: i32) -> i32 {
    let mut  memo= Vec::with_capacity((n + 1) as usize);
    for m in 0..(n +1){
        memo.push(-1);
    }
    memo[0] = 1;
    memo[1] = 1;
    for  i   in 2..(n + 1) as usize{
        memo[i] = memo[i -1] + memo[i -2];
    }
    return memo[n as usize];

}

fn main() {
    println!("{}",climb_stairs(20));
